################# App Blog interactif ###################

Description : plateforme de blog interactif
			écris en PHP, MySql, Doctrine, Jquery

- créez un compte et connectez vous pour continuer !

- onglet "Home" : liste des articles

- l'onglet "WriteArticle", permet d'écrire un article

-l'onglet "MyArticles" liste de vos articles

- sur vos articles, cliquez 'Delete' pour supprimer l'article

- vous pouvez commenter tous les articles, cliquez sur le boutton "read article" 
en bas dans le champ "commenter" ecrivez votre commentaire puis cliquez sur 
le boutton d'envoie à droite "l'icone envoie"



-onglet "se deconnecter" : se deconecter


NB : 
1- creez une base de donnee mysql: onlinesales
2- creer le schema : cmd-> php bin/console orm:schema-tool:create
3- utiliser des fake data (users, articles, commentaires): cmd -> php dump_data.php


vous pouvez egalement utiliser la cmd ->php bin/loader pour charger les donnees et 
lancer l'application en localhost
